# Story 1.3.1: Floating Action Button (FAB) / Report Button

**Epic:** 1.3 60-Second Report Submission Flow  
**Story Points:** 2  
**Sprint:** 1  
**Status:** Not Started

---

## User Story

**As a** user  
**I want** a prominent button to start reporting an issue  
**So that** I can quickly submit a report without navigating through menus

---

## Acceptance Criteria

- [ ] Mobile: FAB (Floating Action Button) in bottom-right corner, 56x56px
- [ ] Desktop: "Report Issue" button in bottom-right corner, standard button size
- [ ] Icon: Camera (Hugeicons `Camera01`)
- [ ] Primary green color (#059669)
- [ ] Tap/click opens report form modal
- [ ] FAB floats above map (z-index: 40)
- [ ] Always visible (no scroll hiding)
- [ ] Accessible label: `aria-label="Report environmental issue"`

---

## Technical Implementation

### Components to Create

- `/components/report/ReportFAB.tsx`
- `/components/report/__tests__/ReportFAB.test.tsx`

### Code Structure

```typescript
'use client';
import { Camera01 } from '@hugeicons/react';
import { useRouter } from 'next/navigation';

export function ReportFAB() {
  const router = useRouter();

  return (
    <button
      aria-label="Report environmental issue"
      onClick={() => router.push('/report/new')}
      className="fixed bottom-6 right-6 z-40 w-14 h-14 bg-green-600 rounded-full shadow-lg hover:bg-green-700 transition-colors md:w-auto md:px-6 md:h-12 md:rounded-lg"
    >
      <Camera01 size={24} color="white" aria-hidden="true" />
      <span className="hidden md:inline ml-2 text-white font-medium">
        Report Issue
      </span>
    </button>
  );
}
```
